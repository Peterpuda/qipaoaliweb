# 视频文件存放目录

## 📁 文件位置

将你的英雄区域背景视频放在这里。

## 📝 文件命名

**必须命名为**: `hero-background.mp4`

## 🎬 视频规格要求

### 推荐规格
```
格式: MP4 (H.264 编码)
分辨率: 1920x1080 (Full HD)
比例: 16:9
帧率: 30 fps
码率: 2-5 Mbps
文件大小: < 10 MB（推荐）
时长: 10-30 秒（循环播放）
音频: AAC 编码, 128 kbps
```

## 🎨 内容建议

- 非遗手工艺场景（刺绣、编织、陶艺等）
- 慢动作特写
- 暗色调（配合深色主题）
- 细节镜头（手部动作、材质纹理）
- 视频首尾要衔接流畅（循环友好）

## 📱 移动端说明

移动端（屏幕宽度 < 768px）会自动禁用视频，显示静态背景，以节省流量和提升性能。

## ⚙️ 使用方法

1. 将视频文件命名为 `hero-background.mp4`
2. 放在当前目录 (`frontend/videos/`)
3. 刷新首页即可看到效果

## 🔧 如果没有视频

如果没有准备视频文件，系统会：
- 显示现有的 poster 图片作为背景
- 或显示原有的渐变背景
- 不会影响页面正常显示

## 🎵 声音控制

- 视频默认静音
- 用户可点击右下角的声音按钮开启/关闭声音
- 可点击播放/暂停按钮控制播放

## 📊 性能优化

- 视频采用懒加载
- 页面不可见时自动暂停
- 移动端不加载视频
- 如果加载失败，自动回退到静态背景
