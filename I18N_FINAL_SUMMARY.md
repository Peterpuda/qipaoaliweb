## 📋 集成清单（剩余页面）

### 快速集成步骤

每个页面只需 3 步：

#### 步骤 1: 添加脚本（30秒）
```html
<head>
  <!-- i18n -->
  <script src="/i18n/index.js"></script>
  <script src="/common/i18n-helper.js"></script>
</head>
```

#### 步骤 2: 添加属性（5-10分钟）
```html
<h1 data-i18n="page.title">标题</h1>
<button data-i18n="common.submit">提交</button>
<input data-i18n-placeholder="common.search" placeholder="搜索">
```

#### 步骤 3: 初始化（1分钟）
```javascript
window.addEventListener('DOMContentLoaded', async () => {
  await initI18n({
    autoDetect: true,
    translateOnInit: true,
    createSwitcher: true,
    switcherContainerId: 'languageSwitcher'
  });
});
```

### 常见问题
1. **语言切换器不显示？**
   - 确保有 `<div id="languageSwitcher"></div>`

2. **翻译不生效？**
   - 检查 i18n 脚本是否加载
   - 检查 data-i18n 属性是否正确
   - 检查翻译 key 是否存在

3. **如何添加新语言？**
   - 在 `i18n/locales/` 添加新语言包
   - 在 `i18n/index.js` 添加语言代码
