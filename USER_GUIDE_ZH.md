## 🎯 新功能介绍

### ✨ 全局导航栏

每个页面顶部都有统一的导航栏，包含：
- **返回按钮** ⬅️ - 返回上一页
- **回主页按钮** 🏠 - 快速返回主页
- **主菜单** - 商城、匠人、治理、签到、奖励
- **语言切换** 🌐 - 支持 7 种语言
- **钱包连接** 💰 - 连接 Web3 钱包

### 🌍 多语言支持

点击导航栏右上角的语言切换器，选择您的语言：
- 🇨🇳 简体中文
- 🇬🇧 English
- 🇯🇵 日本語
- 🇫🇷 Français
- 🇪🇸 Español
- 🇷🇺 Русский
- 🇲🇾 Bahasa Melayu

### 🔐 Admin 管理后台

访问 `/admin/` 会自动跳转到登录页面：
1. 点击"连接钱包并登录"
2. 选择您的钱包（如 MetaMask）
3. 签名验证身份
4. 登录成功后进入管理中心

**注意**：
- 仅授权的管理员钱包可以登录
- 登录有效期为 **7 天**
- 7 天后需要重新登录
- 点击右上角"退出"按钮可以登出

## ⚠️ 注意事项

1. **钱包连接**
   - 首次使用需要安装 MetaMask 或其他 Web3 钱包
   - 连接钱包后会显示简化的地址（如 `0x1234...5678`）

2. **Admin 登录**
   - 只有授权的钱包地址可以登录管理后台
   - 登录有效期 7 天，过期后需要重新登录
   - 退出登录后需要重新签名验证

3. **语言切换**
   - 切换语言后页面会立即刷新
   - 所有内容（包括按钮、提示）都会切换到对应语言
   - 语言选择会保存在浏览器中

4. **导航菜单**
   - 当前页面的菜单项会高亮显示
   - 点击菜单项会跳转到对应页面
   - 移动端点击菜单项后菜单会自动关闭

## 🐛 常见问题

### Q: 导航栏遮挡了页面内容怎么办？
A: 这是正常的，页面已经自动添加了顶部留白（64px），内容不会被遮挡。

### Q: 语言切换器无法点击？
A: 请刷新页面（Ctrl/Cmd + R），确保 JavaScript 已加载。

### Q: Admin 登录失败？
A: 请确认：
- 已安装 MetaMask
- 钱包已解锁
- 您的钱包地址在管理员白名单中

### Q: 钱包连接后无法签名？
A: 请检查 MetaMask 弹窗是否被浏览器拦截，需要允许弹窗。

### Q: 7 天后登录过期怎么办？
A: 再次访问 Admin 页面会自动跳转到登录页，重新连接钱包并签名即可。
